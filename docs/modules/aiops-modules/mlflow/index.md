# Mlflow Modules

This section contains all mlflow modules available in the IDF framework.


??? info "Mlflow AI Gateway Image"
    
    **Module:** `mlflow/mlflow-ai-gw-image`
    
    This module creates a [MLFlow AI Gateway](https://mlflow.org/docs/latest/llms/index.html#id2) server container image and pushes to the specified Elastic Container Repository.
    
    [View Details](./mlflow-ai-gw-image.md){ .md-button }
    [GitHub Source](https://github.com/awslabs/aiops-modules/tree/main/modules/mlflow/mlflow-ai-gw-image){ .md-button .md-button--primary }
